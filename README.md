# typing-speed-test

Measure your typing speed in WPM and accuracy.

Pick a text passage, type it out, see your results. Common passages, code snippets, or random words — three modes.

## Features

- WPM calculated live as you type
- Accuracy percentage
- Error highlighting (incorrect characters shown in red)
- Results history (last 10 tests)
- 1-minute, 2-minute, and custom duration modes

## Run

```bash
npm install && npm run dev
```
